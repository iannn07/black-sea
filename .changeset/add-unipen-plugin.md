Add UNIPEN (`skills/unipen/`), a new skill that generates GDS-standard project documentation
(Onboarding, Technical Documentation, Release Notes, Code Documentation) into an Obsidian vault.
UNIPEN is a fully separate job from the BLACK SEA investigation unit — no shared files or
vocabulary, like BULKHEAD — and is bundled into the single `black-sea` plugin alongside it.
