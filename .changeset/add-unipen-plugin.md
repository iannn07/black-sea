Add UNIPEN, a new independently-installable plugin in this marketplace that generates
GDS-standard project documentation (Onboarding, Technical Documentation, Release Notes, Code
Documentation) into an Obsidian vault. UNIPEN lives at `unipen/` (source `./unipen`), is a fully
separate job from the BLACK SEA investigation unit, and installs independently of it — adding
this marketplace and installing only `unipen` pulls in none of BLACK SEA's skills, and vice
versa.
